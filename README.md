# PCB Designs

A collection of printed circuit board (PCB) designs for various projects.

## Overview

This repository contains PCB design files, schematics, and documentation for custom electronic circuits.

## Getting Started

### Prerequisites
- [KiCad](https://www.kicad.org/) - Open source EDA software

### Manufacturing
- [JLCPCB](https://jlcpcb.com/)
